The tiny turtle note taker.

## install

```
npm install -g shellnotes
```

## example usage

```
shellnotes Something I want to take a note on
```

Open today's note

```
shellnotes --today
```

See all of your notes

```
shellnotes --all
```

You can also use the command `note` for shorthand instead of `shellnotes`# shellnotes
